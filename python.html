Einführung

Python ist eine Programmiersprache, die häufig in Webanwendungen, Softwareentwicklung, in der Datenwissenschaft und im Machine Learning (ML) verwendet wird. Entwickler benutzen Python, weil es effizient und leicht zu erlernen ist und auf vielen verschiedenen Plattformen laufen kann.

Zu den Vorteilen von Python gehören:

  • Entwickler können ein Python-Programm leicht lesen und verstehen, da es eine grundlegende Syntax hat
  • Python verfügt über eine große Standardbibliothek, die wiederverwendbare Codes für fast jede Aufgabe enthält. Dadurch müssen Entwickler keinen Code komplett neu schreiben.
  • Entwickler können Python problemlos mit anderen gängigen Programmiersprachen wie Java, C und C++ verwenden.
  • Python ist auf verschiedene Computer-Betriebssysteme wie Windows, macOS, Linux und Unix portierbar.
Hello World

Wenn wir versuchen, eine neue Programmiersprache zu lernen, fangen wir oft damit an, zu lernen, wie man die Anweisung „Hello World“ aus dem Bildschirm ausgibt.

Das ist eine ziemlich einfache Aufgabe. Die einzige Voraussetzung, die du dafür kennen musst, ist die Python print-Anweisung. Diese Anweisung gibt alles aus, was ihr übergeben wurde.

print("Hello World")

Dieser Codeblock sorgt wegen der print-Anweisung für die Ausgabe "Hello World" auf der Konsole.

Variablen

Eine Variable im allgemeinsten Sinne ist einfach ein Behälter (Container) zur Aufbewahrung von bestimmten Werten, also z.B. Strings oder Zahlen. Man kann im Verlauf des Programms auf diese Variablen, oder genauer auf den Wert ihres Inhaltes zugreifen, oder ihnen einen neuen Wert zuweisen.

i = 42

Obige Anweisung darf man nicht als mathematisches Gleichheitszeichen sehen, sondern als "der Variablen i wird der Wert 42 zugewiesen", d.h. der Inhalt von i ist nach der Zuweisung 42. Man kann diesen Wert der Variablen auch, wie man im folgenden Beispiel sieht, anschließend ändern:

i = i + 5

Nun hat die Variable i den Wert 47. Anders als in den meisten Programmiersprachen, muss man in Python den Datentyp einer Variable nicht explizit angeben. So wird die Eingabe "42" automatisch als Ganzzahl erkannt.

if-Bedingung

In Python gibt es die Möglichkeiten Bedingungen zu überprüfen und entsprechend im Programmablauf darauf zu reagieren. Damit das Programm fehlerfrei abläuft, muss der Code, welcher zu der if-Bedingung gehört, unbedingt eingerückt werden.

wert = 7
if wert < 5:
    print("Ich werde nur ausgeführt, wenn 7 < 5 gilt.")
print("Ich werde unabhängig von der Bedingung ausgeführt, denn ich bin nicht eingerückt.")

In diesem Beispiel wird die erste print-Anweisung nur ausgegeben, wenn die Bedingung stimmt. Manchmal ist es jedoch auch von Vorteil, wenn bestimmter Code nur ausgeführt wird, wenn die Bedingung nicht zutrifft. In diesem Fall kann man Gebrauch von dem Wort else machen.

wert = 7
if wert < 5:
    print("Ich werde nur ausgeführt, wenn 7 < 5 gilt.")
else:
    print("Ich werde nur ausgeführt, wenn 7 < 5 nicht gilt.")
print("Ich werde unabhängig von der Bedingung ausgeführt, denn ich bin nicht eingerückt.")
Datentypen

In Python gibt es 6 grundlegende Datentypen, die oft verwendet werden. Diese lauten:

  • integer:Ganzzahlen, z. B. 6
  • float:Fließkommazahlen, z. B. 1,5
  • string:Zeichenketten, z. B. "Hello World"
  • boolean:Wahrheitswerte, z. B. True oder False
  • list:Liste, z. B. [“Hose“, “Hemd“]
  • tuple:Tupel (nicht veränderbare Liste), z. B. (‘Pizza‘,‘Salat‘,‘Eis‘)
Schleifen

Schleifen, werden benötigt, um einen Codeblock, den man auch als Schleifenkörper bezeichnet, wiederholt auszuführen. In Python gibt es zwei Schleifentypen: die while-Schleife und die for-Schleife.

Die for-Schleife wird hauptsächlich auf 2 verschiedene Arten angewendet.

for wert in meine_liste:
    print("Tu etwas mit wert")

for index in range(x):
    print("Mach etwas x-mal (x ist eine Ganzzahl)")

In der ersten for-Schleife wird eine Liste mit dem Namen "meine_liste" durchlaufen. Die Variable "wert" nimmt dabei den Wert des entsprechenden Listen-Elements an. Im ersten Schleifen-Durchlauf hat die Variable den Wert des ersten Listen-Elements, beim letzten Durchlauf den des letzten Elements. Erst danach wird mit dem Code außerhalb der Schleife (nicht eingerückt) fortgesetzt. In der zweiten for-Schleife wird der Code innerhalb der Schleife x-mal ausgeführt. "x" kann dabei auch als die Länge einer Liste (Anzahl der Elemente) definiert werden, auch wenn diese nicht bekannt ist.

x = 4
while x < 5:
    print("Tu etwas, solange x < 5 gilt.")

Die while-Schleife wird ausgeführt, wenn die angegebene Bedingung stimmt. Anders als bei der for-Schleife, weiß man nicht, wie viele Durchläufe gemacht werden. Bei einem fehlerhaften Schleifenkörper kann es zu sogenannten Endlos-Schleifen kommen. Dabei wird im Schleifenkörper die eigentliche Bedingung nicht verändert, wodurch die Schleife unendlich oft wiederholt werden würde. Daher muss im Beispiel die Variable "x" innerhalb des Schleifenkörpers so verändert werden, dass sie irgendwann größer oder gleich 5 ist, wodurch die Bedingung nicht mehr zutrifft und die Schleife nicht mehr ausgeführt wird.

Funktionen

Das Konzept einer Funktion ist eines der wichtigsten in der Mathematik. Eine übliche Verwendung von Funktionen in Computersprachen ist die Implementierung mathematischer Funktionen. Eine solche Funktion berechnet ein oder mehrere Ergebnisse, die vollständig durch die an sie übergebenen Parameter bestimmt werden.

def quadrieren (x):
    z = x * x
    return z

Im Beispiel bekommt die Funktion "quadrieren" einen Parameter ("x"). Der Rückgabewert dieser Funktion ist das Quadrat des Parameters. Möchte man nun das Quadrat einer Zahl bekommen, so muss man nur die Funktion aufrufen.

a = quadrieren(4)

Auf diese Art kann die Länge des Codes verkürzt werden und eine Dopplung von Codeblöcken (schlechter Programmier-Stil) vermieden werden.
